## Problem Statement

El juego tiene tres áreas a mejorar:
1. Las pistas de habitación dicen "estaba en Cocina" → debe decir "estaba en la cocina".
2. Los objetos del tablero usan nombres genéricos que no encajan con la estética. Sustituir por iconos y nombres de mansión.
3. El tablero más grande es 7×7. Evaluar y agregar 8×8, 9×9, 10×10.

## Premises (confirmed)

1. IDs internos ('cama', 'mesa', 'tv', 'estanteria', 'caja') NO cambian.
2. 'cama'/'banco' sigue en OCCUPIABLE_OBJECTS.
3. Ventana→Cuadro es solo display (clueText + BoardGrid CSS). Clue type `next_to_window` intacto.
4. Artículos solo en pistas (etiquetas del tablero quedan capitalizadas).
5. Tableros grandes solo si benchmark < 40s.

### Tarea C — Tableros 8×8, 9×9, 10×10

**Benchmark (PRIMERO):**
```bash

bun scripts/benchmark-sizes.ts
# Si 8×8 < 40s → incluir. Si 9×9 < 40s → incluir. Si 10×10 < 40s → incluir.
```

Script `scripts/benchmark-sizes.ts`:
```typescript
import { generatePuzzle } from '../src/logic/generator'
for (const [rows, level] of [[8, 21], [9, 26], [10, 31]]) {
  const t0 = Date.now()
  for (let i = 0; i < 3; i++) generatePuzzle(level, i)
  console.log(`${rows}×${rows}: avg ${Math.round((Date.now()-t0)/3)}ms`)
}
```

**Si pasan benchmark → `src/logic/generator.ts`:**
```typescript
function getDifficultyConfig(level: number, rng: () => number): BoardConfig {
  if (level <= 10) return { rows: 5, cols: 5, numChars: 4, numRooms: seededInt(3, 4, rng) }
  if (level <= 15) return { rows: 6, cols: 6, numChars: 5, numRooms: seededInt(4, 5, rng) }
  if (level <= 20) return { rows: 7, cols: 7, numChars: 6, numRooms: seededInt(4, 5, rng) }
  if (level <= 25) return { rows: 8, cols: 8, numChars: 7, numRooms: seededInt(5, 6, rng) }  // if 8×8 passes
  if (level <= 30) return { rows: 9, cols: 9, numChars: 8, numRooms: seededInt(5, 6, rng) }  // if 9×9 passes
  return { rows: 10, cols: 10, numChars: 9, numRooms: seededInt(6, 7, rng) }                  // if 10×10 passes
}
```

ROOM_NAMES tiene 10 entradas → suficiente para hasta 7 habitaciones por tablero.
