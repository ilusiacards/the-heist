# Plan: Multi-Cell Objects — Banco y Alfombra 2×1 (v0.3.0)

Generated by /office-hours + /autoplan on 2026-05-29
Branch: main
Status: IN REVIEW

## Problem Statement

Los tableros 8×8 y 9×9 tienen espacio para objetos más grandes. Banco (`cama`) y Alfombra (`alfombra`) pueden aparecer en versión 2 casillas (horizontal o vertical), haciendo el mapa más realista y añadiendo dificultad en tableros grandes.

## Premises (confirmed by user)

1. Solo `cama` y `alfombra` pueden ser 2×1 (resto siempre 1×1).
2. La versión 2×1 es probabilística (aleatoria, escala con tamaño de tablero).
3. Pistas sin cambio: ambas casillas del span = "está en el objeto".
4. IDs internos intactos (`cama`, `alfombra` no cambian).
5. Un solo SVG renderizado desde la celda primaria con dimensión doble.
6. Modelo de datos: `objectSpanDir?: 'h' | 'v'` en celda primaria; `objectPartOf?: CellId` en celda secundaria.

### T2 — Generador: función `applySpanUpgrades` (`src/logic/generator.ts`)

Nueva función llamada al final de `generateBoard()`, DESPUÉS del bucle de garantía de ocupabilidad:

```typescript
const SPAN_PROB: Record<number, number> = { 5:0, 6:0.20, 7:0.35, 8:0.50, 9:0.60 }

function applySpanUpgrades(cells: Cell[][], rows: number, cols: number, rng: () => number): void {
  const prob = SPAN_PROB[rows] ?? 0
  if (prob === 0) return
  for (let r = 0; r < rows; r++) {
    for (let c = 0; c < cols; c++) {
      const cell = cells[r]![c]!
      if (cell.object !== 'cama' && cell.object !== 'alfombra') continue
      if (cell.objectPartOf) continue   // ya es secundaria de otro span
      if (rng() > prob) continue
      // Intentar H (derecha) y V (abajo) en orden aleatorio
      const dirs: Array<[number, number, 'h' | 'v']> = []
      if (c + 1 < cols) dirs.push([r, c + 1, 'h'])
      if (r + 1 < rows) dirs.push([r + 1, c, 'v'])
      if (rng() < 0.5) dirs.reverse()
      for (const [nr, nc, dir] of dirs) {
        const neighbor = cells[nr]![nc]!
        if (neighbor.object !== undefined) continue   // celda ocupada
        if (neighbor.roomId !== cell.roomId) continue // diferente habitación
        cell.objectSpanDir = dir
        neighbor.object = cell.object
        neighbor.objectPartOf = cell.id
        break
      }
    }
  }
}
```
